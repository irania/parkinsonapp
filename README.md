# ParkinsonApp

A Unity project for a Parkinson-related application, built with Unity and using Unity's package and project settings.

## Overview

This repository contains a Unity project with assets, scripts, scenes, and configuration for an interactive application.

## Project Structure

- `Assets/` - Unity asset files, scripts, scenes, prefabs, textures, audio, and other game data.
- `Packages/` - Unity package manifest files.
- `ProjectSettings/` - Unity editor and project configuration settings.

## Getting Started

1. Install Unity Hub and the Unity version used by this project.
2. Open Unity Hub and add the project folder:
   - `c:\Users\atefe\Documents\GitHub\parkinsonapp`
3. Open the project in the Unity Editor.

## Notes

- The repository does not currently include a dedicated README or setup scripts beyond the Unity project files.
- Use Unity Editor to manage scenes, builds, and assets.
